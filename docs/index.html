ng2-mouse-select

Description

This library has following features:

  • sets a app-selectFrame component within which you can define selectable components with the help of 'selectable' directives. (example down)

  • it allows filtering of selection based on configurable properties(with the use of 'ensureSame' input), so that only items with the same value(s) of these properties get selected.

  • with the configuration of the 'filter' you can determine on which elements will the directive be active, that is, on which elements can you start the selection (for instance, you can configure the selection to be active just on the app-selectFrame, or you can add 'appAllowSelect' directive to the element on every element you want the selection to be active).

  • setting a 'selectable' directive to a component inside the 'app-selectFrame' enables you to define data which will be returned from the component after the selection ('select' input).

Installation

To install this library, run:

$ npm install ng2-mouse-select --save

Consuming your library

Once you have published your library to npm, you can import your library in any Angular application by running:

$ npm install ng2-mouse-select

and then from your Angular Module:


// Import your library
import { SelectableModule } from 'ng2-mouse-select';

@NgModule({
    ...
    imports: [
      ...
        SelectableModule
    ],
   ...
})

Once your library is imported, you can use its components and directives and pipes in your Angular app:

<!-- You can now use your library component in component.html -->
<!-- Example: -->
 <div app-selectFrame [ensureSame]="['property1','property2']"
         [filter]="{ allowSelectElements: true, thisElement: true }" 
         (data)="onMousePalletsSelected($event)">

<div class="col-md-6" appAllowSelect>
....
 <tr some-component [someClass]="someClass" 
     selectable
     [select]="{'property1':someClass.data1,'property2':someClass.data2}">
  </tr>
....
</div>
